// QUEUE MEMORY MODEL

  • Queue 為線性陣列，front 從低位址端出列，rear 從高位址端入列
  • front = 0, rear = -1 表示空 Queue；rear = N-1 表示已滿
  • enqueue：q[++rear] = x · dequeue：x = q[front++]
  • Dequeue 後舊值以殘值留在記憶體；本實作為 Linear Queue（非循環）
