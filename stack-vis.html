// STACK MEMORY MODEL

  • Stack 底部在低位址 (stk[0])，向高位址生長
  • top = -1 表示空 Stack；top = N-1 表示已滿
  • push：stk[++top] = x · pop：x = stk[top--]
  • Pop 後舊值仍殘留在記憶體，直到下次 Push 覆蓋
