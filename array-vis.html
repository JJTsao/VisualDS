記憶體配置說明

📌 int list[5] 宣告後，系統在 Stack 上配置連續的 20 bytes。

📌 每個 int 佔 4 bytes，位址間距固定為 4。

📌 list[i] 等同於 *(list + i)，即 base address + i × 4。

⚠️ C++ 不做越界檢查，越界存取屬 Undefined Behavior。
